Function Details

writehash(hashfile, logfile, hash_size=0)

source code 

Write a hash file based on the contents in the log file.

If the log file hasn't been changed since the existing hashfile was created, this is a no-op.

Parameters:
  • hashfile - file to create. If it already exists, it will atomically be updated.
  • logfile - file to read from. It must exist.
  • hash_size - Valid values are 0, 4, 8. 0 means autoselect hash size. 4 is 32 bit hash, 8 is 64 bit hash.
