__init__(self, hashfile, logfile, mode='NEW', compression_type=0, compression_block_size=0, hash_size=0)
(Constructor)

source code 

Creates a new writer.

Does everything that LogWriter does, but also writes the hash file.

Parameters:
  • hashfile - filename of hash file
  • logfile - filename of log file
  • mode - Same as in LogWriter.__init__
  • compression_type - Same as in LogWriter.__init__
  • compression_block_size - Same as in LogWriter.__init__
  • hash_size - Valid values are 0, 4, 8. 0 means autoselect hash size . 4 is 32 bit hash, 8 is 64 bit hash.
